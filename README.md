# BioMEDirect static site

This repository contains a static site with Biology, Genetics, and Embryology lessons. The entry point is `index.html`, which links to the lesson menu at `TextT/CentralMenuT.html`.

## Live preview
Use any static HTTP server to view the site locally. For example, with Python installed:

```bash
cd /workspace/BioMEDirect
python -m http.server 3000
```

Then open <http://localhost:3000> in your browser to see the landing page. Click the splash image to reach the main menu.
